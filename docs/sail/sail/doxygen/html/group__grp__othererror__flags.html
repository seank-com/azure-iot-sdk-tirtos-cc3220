§ PGA460_CHECKSUMERRORFLAG

#define PGA460_CHECKSUMERRORFLAG   0x0100U

The error status returned on the register read or response command will be a uint16. The 8th bit tells whether there was a checksum error. if the bit is set, then there was a checksum error.
