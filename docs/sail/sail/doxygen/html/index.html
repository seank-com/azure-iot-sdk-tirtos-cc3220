SAIL API Guide

Introduction

SAIL the sensor and actuator interface layer provides software modules for interfacing with sensors, actuators, input devices and display devices like TMP007, OPT3001, push buttons etc. The SAIL modules use TI drivers for interfacing. The SAIL modules are scalable across devices MSP432, CC3220, CC2640R2, CC1310 and can be used with the SimpleLink SDK seamlessly. The below figure shows how SAIL fits into the SDK ecosystem.

Salient features of SAIL modules

  1. Written completely in C.
  2. Follows well defined coding guidelines.
  3. Detailed API user guide is provided for each module.
  4. Easy to understand, and as self-contained as possible.
  5. SAIL modules support multiple tool chains.
  6. Provides console based examples which show the usage of various API of SAIL modules.
  7. Provides few complex examples which use various SAIL modules to implement a BoosterPack Example.
