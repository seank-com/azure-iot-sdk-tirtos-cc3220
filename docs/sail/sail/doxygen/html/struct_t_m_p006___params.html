Detailed Description

TMP006 Parameters.

TMP006 parameters are used with the TMP006_open() call. Default values for these parameters are set using TMP006_Params_init(). The GPIO_CallbackFxn should be defined by the application only if the data ready functionality is desired. A gpioIndex must be defined in the TMP006_HWAttrs for the corresponding tmp006Index. The GPIO_CallbackFxn is in the context of an interrupt handler.
