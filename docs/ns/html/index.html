Network Services API Reference

Modules

The Network Services product includes support for the following application layer networking protocols:

  • HTTP Client - Provides a HTTP/1.1 interface for client application development.
  • HTTP/2 Client (experimental) - Provides a HTTP/2 interface for client application development.
  • SNTP Client - Provides a SNTP interface to synchronize local time with NTP servers.
  • TLS - Provides a portable TLS interface to create/delete TLS contexts for SimpleLink WiFi and WolfSSL TLS layers.
